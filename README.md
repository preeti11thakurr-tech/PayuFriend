# 💸 PayuFriend

A Full Stack Digital Wallet Application built using **React.js, Node.js, Express.js and MongoDB**.

---

## 🚀 Features

- 🔐 Secure Signup & Login using JWT Authentication
- 🔑 Password Encryption using bcrypt
- 💰 Wallet Balance
- 💸 Transfer Money
- 📜 Transaction History
- 🎨 Modern Responsive UI

---

## 🛠 Tech Stack

### Frontend

- React.js
- Vite
- Axios
- CSS

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/preeti11thakurr-tech/PayuFriend.git
```

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
